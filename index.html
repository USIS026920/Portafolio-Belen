Mis Proyectos

Voto nación:

La app voto nación está hecha para un conteo de votos en tiempo real, además integra un servicio de chat en tiempo real de forma general, también cuenta con el uso de graficas para causar una mejor experiencia al usuario.

Clínica médica:

La aplicación es de una clínica médica, esta aplicación es para el área de la salud, ya que esta aplicación cuenta con una serie de opción como: recetas médicas, inventario, citas, diversos registros, registros especiales, para que el usuario pueda relacionarse con ellas para que se le haga más fácil el trabajo.

Casa de nieve en 3D:

Consiste en modelar una casa en 3D haciendo uso del programa de Blender, donde vamos aplicar varios materias (arboles, nieve, luces, colores, etc).

Robocar:

Competencia de carritos, consiste en una carrera de carritos, realizado en la UGB.

Pista de baile en 3D:

Consiste en animar un personaje en 3D haciendo uso del programa de blender, y de una web que proporciona personajes y animaciones.

Travel Accident:

Es una aplicacion web, esta aplicación es para el área de transito, ya que está diseñada para el control de accidentes de transito en El Salvador.

Iris accesorios:

La plataforma en línea permite a los usuarios visualizar y comprar productos de manera eficiente y segura. El sitio web fue diseñado con una interfaz intuitiva y accesible, facilitando la navegación y mejorando la experiencia de usuario.
